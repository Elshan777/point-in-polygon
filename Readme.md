# Point in a polygon

This is a school project that randomly creates points and calculates if a point is inside or outside of a polygon

# Demonstration

![](demo.gif)

## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install shapely and tkinter.

```bash
pip install shapely
pip install tkinter
```
